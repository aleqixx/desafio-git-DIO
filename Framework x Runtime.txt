FRAMEWORK = É COMO SE FOSSE UMA CAIXA DE FERRAMENTAS COM FERRAMENTAS PRONTAS PERMITINDO
AO DEV UTILIZÁ-LAS COMO BEM ENTENDER

FRAMEWORK 'BOOTSTRAP': TRAZ PRONTO, FUNCIONALIDADES BARRA DE PORCENTAGEM, BOTÃO DE PERIGO
OU ALERTA, POUPANDO AO DEV O TRABALHO DE TER QUE CRIAR ESTAS FUNÇÕES DO ZERO, TALVEZ
APENAS FAZER PEQUENAS ALTERAÇÕES DE CORES E TAMANHOS, CONFORME A NECESSIDADE

********** RUNTIME **********

DE PROGRAMAÇÃO PARA A LINGUAGEM BINÁRIA, FAZENDO COM QUE O COMPUTADOR ENTENDA O BLOCO
DE CÓDIGO CRIADO E RETORNA O RESULTADO ESPERADO
